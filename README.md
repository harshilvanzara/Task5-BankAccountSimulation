💰 Bank Account Simulation (Java OOP)

📌 Objective

Simulate basic bank operations using Java OOP concepts.

🛠 Tools Used

- Java
- VS Code
- Terminal

🚀 Features

- Deposit money
- Withdraw money
- Check balance

📂 Project Structure

- BankAccount.java → Handles account operations
- Main.java → Runs the program

▶️ How to Run

javac BankAccount.java Main.java
java Main

- Classes & Objects
- Encapsulation
- Methods
